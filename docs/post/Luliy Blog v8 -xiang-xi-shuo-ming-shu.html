📄 JSON 配置文件说明

常见配置文件结构

{
  "site": {
    "title": "Luliy Blog",
    "description": "个人博客",
    "author": "Your Name",
    "url": "https://your-domain.com"
  },
  "theme": {
    "primaryColor": "#6366f1",
    "darkMode": true,
    "themes": ["light", "dark", "sakura", "midnight"]
  },
  "modules": {
    "particles": false,
    "audioSFX": true,
    "sakuraPetals": true,
    "darkModeRipple": true
  },
  "navbar": {
    "sticky": true,
    "glassEffect": true,
    "transparent": false
  },
  "timeline": {
    "enabled": true,
    "groupBy": "year"
  }
}

关键配置项修改指南

配置项 类型 说明 修改例
primaryColor String 主色调 "#ff6b6b"
theme Array 启用主题列表 ["light", "dark"]
audioSFX Boolean 启用音效 false
sakuraPetals Boolean 樱花效果 true
uptimeStart String 正常运行起始日期 "2024-01-01"
