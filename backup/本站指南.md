这份 JSON 文件是基于 **Gmeek** 框架（一个利用 GitHub Issues 自动化构建的个人博客系统）的超级魔改版配置文件（通常命名为 `config.json`）。

这份配置中融入了大量的自定义 CSS 样式、前端交互脚本以及酷炫的特效（如樱花飘落、粒子点击、打字机、彩带纸屑等）。

为了让你彻底搞懂，我把这段代码拆解为以下 6 个核心板块来逐一解析：

---

## 1. 基础元数据与全局配置

这部分定义了博客的基本身份信息和系统运行环境：

* **`title` / `displayTitle**`: 博客的总名称，这里叫 `"Luliy"`。
* **`subTitle`**: 博客的副标题（签名）：`"我将无限进步!"`。
* **`avatarUrl` / `faviconUrl` / `ogImage**`: 你的头像、浏览器标签页小图标、以及网页分享时显示的缩略图链接。
* **`GMEEK_VERSION`**: 规定使用的 Gmeek 框架版本，这里设置为自动获取最新版 (`"last"`)。
* **`homeUrl`**: 博客的主页网址：`https://luliy6.github.io`。
* **`email`**: 你的联系邮箱：`luliy6@qq.com`。
* **`startSite`**: 博客的建站日期：`05/30/2026`。
* **`onePageListNum`**: 首页每页文章显示的条数（一页显示 7 篇）。
* **`singlePage`**: 独立页面的白名单列表，包含关于我、朋友、画廊、工具、时间轴、音乐、书籍、提示词和归档。

---

## 2. 图标与外部链接

* **`iconList`**: 里面包含了一整套 SVG 地图路径数据。它负责在博客的各个导航标签（如 book, prompt, archive, link 等）前面绘制出扁平化的精致小图标。
* **`exlink`**: 外部链接映射表。当你在博客点击 GitHub、邮件、RSS、标签时，系统会自动去读这里配置的跳转 URL。

---

## 3. 功能功能性开关与偏好设置

* **`i18n`**: 语言设为 `"CN"`（中文）。
* **`UTC`**: 时区设为 `8`（北京时间）。
* **`urlMode`**: 文章链接生成模式为 `"pinyin"`（用拼音做 URL 路径，更利于 SEO）。
* **`themeMode`**: 主题切换模式为 `"manual"`（手动切换深色/浅色模式）。
* **`dayTheme` / `nightTheme**`: 白天使用 GitHub 官方的 `light` 浅色主题，夜间使用 `dark_colorblind`（暗黑红绿色盲友好）主题。
* **`yearColorList`**: 归档页面不同年份标题的颜色循环池。
* **`commentLabelColor`**: 评论系统标签的颜色。
* **`showPostSource` / `needComment**`: `1` 表示开启显示文章源码、开启评论功能。
* **`rssSplit`**: RSS 订阅的分隔策略设为 `"sentence"`（按句子切分）。
* **`bottomText`**: 网站底部的版权声明：`"转载请注明出处"`。

---

## 4. 全局头部注入与依赖库 (`primerCSS`, `allHead`)

这是魔改的第一步，往博客的 `<head>` 标签中塞入了大量前端美化和渲染引擎库：

* **`primerCSS`**: 引入了 GitHub 官方的 UI 样式库 Primer CSS，保证了整体风格的一致性。
* **`allHead` 中引入的第三方外部组件**：
* **KaTeX**: 用于在文章中完美渲染复杂的数学公式。
* **GmeekTOC / GmeekVercount**: 文章目录插件和网站访问量统计插件。
* **PrismJS**: 强大的代码高亮渲染器。
* **Mermaid**: 允许你在文章中直接用代码画流程图、时序图。
* **Chart.js**: 数据图表渲染库。
* **APlayer**: 极其精致的网页音乐播放器。


* **`allHead` 中的自定义全局样式**：
* 强制全站使用非常优雅的 **“霞鹜文楷 (LXGW WenKai)”** 字体。
* 定义了平滑滚动 (`scroll-behavior: smooth`) 以及金黄色的文本选中特效。
* 重写了夜间模式下的代码块颜色（暗红代码）、表格边框颜色、以及美化了滚动条。
* **注入了站内搜索功能 (`initSearch`)**：只要在网页上按下 `/` 键或者点击搜索按钮，就会弹出一个毛玻璃效果的文章搜索面板。



---

## 5. 文章页魔改脚本与样式 (`script`, `style`)

### 📜 `style`（文章页专用样式）

专门用来美化具体文章页（`#postBody` 内部）的排版：

* 为 `h2` 标题自动添加**数字序号**，并左置一条粗蓝线和淡淡的浅蓝背景框。
* 将加粗文本 (`<strong>`) 和超链接 (`<a>`) 统一改为显眼的金黄色。
* 升级了引用块 (`blockquote`)，在其左上角加了一个巨大的、半透明的优雅双引号 (`“`)。
* 为表格添加了斑马纹阴影（奇偶行颜色交替）和鼠标悬停高亮特效。
* 设计了归档页面（`archive`）专属的时间轴样式，把文章串成一条漂亮的垂直脉络。

### ⚙️ `script`（文章页增强逻辑）

当读者点进一篇文章时，这段庞大的 JS 脚本会在后台默默执行以下功能：

1. **顶部进度条**：随页面向下滚动，最顶部会延伸出一条蓝色的阅读进度条。
2. **回顶部 + 百分比圆形按钮**：右下角生成一个圆形图表，内部展示当前阅读百分比（如 45%），点击直接平滑回滚到顶部。
3. **代码块增强**：自动检测代码语言并在右上角贴上标签，同时额外生成一个 `⛶` **全屏展开按钮** 和一个 **复制按钮**。
4. **外链保护**：所有的外部链接自动加上 `target='_blank'`，让用户点击外链时在独立新标签页打开，防止流失读者。
5. **阅读数据计算**：根据文章总字数自动计算并置顶提示：“*预计阅读：约 X 分钟 | 共 X 字*”。
6. **锚点复制**：鼠标点击文章的任意标题（H1/H2/H3），会自动复制该标题的直达链接到剪贴板。
7. **无限进步赞赏面板**：在文末生成一个“✨ 和作者无限进步”的华丽渐变按钮，点击会像抽屉一样平滑展开你的打赏二维码（`me.jpg`）。
8. **上一篇 / 下一篇导航**：自动请求后台数据，在文章末尾生成无缝衔接的相邻文章导流卡片。
9. **TOC 滚动自动高亮**：侧边栏目录会随着你屏幕滚动的具体段落，自动将对应的目录项标金加粗。
10. **防读者走失标题**：当你切换到别的浏览器标签页时，博客标题会变成 `👀 别走呀，我还在进步！`；切回来时显示 `✨ 欢迎回来！`。
11. **建站时间精确到秒**：在网页最底部实时渲染一个计时器，告诉你这个网站已经平稳运行了多少天、多少小时、多少分、多少秒（秒数用红色加粗强调）。

## 6. 首页与归档页特效脚本与样式 (`indexStyle`, `indexScript`)

### 📜 `indexStyle`（首页卡片样式）

* 把普通的文章列表重构成悬浮卡片 (`.post-item`)。
* 运用了 `4n+1` 循环选择器，让不同的文章卡片左侧边框呈现**紫、蓝、粉、黄四色交替**。
* 鼠标悬停在卡片上时，卡片会触发丝滑的向上浮动动画，并亮起一道绚丽的渐变外边框。
* 美化了首页“一言（Hitokoto）”金句展示框。

### ⚙️ `indexScript`（首页大卡司特效）

这个脚本堪称视觉炸弹，负责在首页渲染各种大制作动画：

1. **全局公告栏**：置顶一个显眼的蓝色公告：`📢 欢迎来到 Luliy 的博客！……`。
2. **Hero 巨幕星空顶**：在首页顶部生成一个深蓝到极光紫的渐变巨幕，带有微弱科技感的网格纹理，并嵌入了一个**实时同步的数字时钟（时:分）**。
3. **每日一言 (Hitokoto)**：动态调用外部 API，每次刷新首页都会随机展示一句富有哲理的动漫/文学名言。
4. **首图动态抓取器**：
* 这是个非常厉害的逻辑。首页的文章卡片本来是没有图片的。
* 这个脚本会在后台悄悄请求每一篇文章的 HTML，**抓取文章里的第一张图片**作为首页卡片的缩略图封面！
* 如果文章没图，则会自动生成一个炫彩渐变背景，并随机塞入一个 Emoji 占位（如 `✨`,`🚀`,`💡`）。


5. **打字机特效**：让你的博客副标题（Luliy ✦ 我将无限进步…）像有人正在敲击键盘一样，逐字打印出来。
6. **樱花飘落系统 (重点)**：
* 利用 HTML5 `Canvas` 在背景层绘制了 38 片不停随风摇曳、打转、下落的**粉嫩樱花瓣**。
* 左下角贴心地设计了一个 `✨` 悬浮小图标。读者要是觉得晃眼，点一下就能**彻底关闭/开启**樱花特效。


7. **鼠标点击粒子特效**：鼠标在网页上任意地方点击时，会以鼠标为中心向四周爆裂喷射出 18 颗五彩缤纷的**彩色粒子火花**并逐渐淡出。
8. **开门红彩带纸屑 (Confetti)**：每当用户第一次加载或者刷新进入首页，屏幕上方会倾泻而下无数五颜六色的**庆祝彩带碎纸屑**，缓缓飘落，极具仪式感。
9. **自动化时间轴构建（Archive 归档页）**：如果检测到用户当前在 `archive`（归档）页面，脚本会自动请求文章列表 JSON，按照“年份”归类，瞬间渲染出一个极其规整漂亮的编年体文章时间轴。
