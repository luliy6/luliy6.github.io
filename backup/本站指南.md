## 系统概述

Luliy 博客是基于 **Gmeek** 框架的高度定制化个人博客系统。Gmeek 是一个利用 GitHub Issues 作为后端数据源、GitHub Pages 作为前端展示的轻量级静态博客生成器。

### 核心优势

- **零数据库** — Issues 即数据源，纯静态生成，极速加载
- **版本控制** — 所有内容用 Git 管理，天然支持历史追踪
- **无服务器** — 直接部署到 GitHub Pages，无需维护服务器
- **高度定制** — 通过三个配置文件完全控制外观和交互

### 技术栈

| 组件 | 技术 |
|------|------|
| **后端框架** | Gmeek (GitHub Issues + GitHub Pages) |
| **样式** | CSS 3（Grid、Flexbox、自定义属性） |
| **脚本** | 原生 JavaScript（无框架依赖） |
| **字体** | LXGW WenKai（开源中文字体） |
| **音效** | Web Audio API |
| **动画** | CSS Transforms + JavaScript RAF |

## 全站功能详解

### 🔍 站内搜索

**触发方式：**
- 点击顶部右侧浮动按钮 `⌕`（Search）
- 按 `/` 快捷键（输入框外）

**搜索流程：**
```
1. 加载 postList.json
   ↓
2. 实时字符匹配（不区分大小写）
   • 搜索范围：标题 + 描述 + 标签 + 日期
   ↓
3. 显示最多 30 条结果
   • 匹配文章计数：N / Total
   ↓
4. 点击结果卡片跳转到文章页
```

**UI 细节：**
- 搜索框上方显示统计 ("显示 N 篇" / "匹配到 N 篇")
- 无结果时显示 "没有找到匹配内容"
- 结果卡片包含：标题（蓝色）+ 描述（灰色）+ 标签（紫色小标签）

- 活跃标题：黄色背景 + 红色文字 + 左边条纹

**滚动高亮：**

**平滑滚动：**

   ```markdown
   ## 章节 1

   ```

3. **触发部署**
   - Issue 发布后，GitHub Actions 自动运行 Gmeek 构建脚本
   - 约 30 秒后，新文章出现在博客首页

### 🎨 自定义配置

| 配置项 | 作用 | 例子 |

修改 CSS 变量或选择器来改变：
- 颜色（所有 `#8250df` 替换为新色）
- 间距（padding、margin）
- 字体（font-family 对应部分）

- 音效（playSfx 的 frequency/gain 参数）

   - 上传新的 `enhance.js` 到 `/` 目录

   - 约 1-2 分钟后部署完成
   - 访问博客 URL 查看效果

## 故障排查

### 问题 1：目录为空或不显示

**原因：** 文章中没有 H2/H3 标题，或 Gmeek TOC 插件未生成

**解决方案：**
```markdown
# 这是 H1（不会出现在 TOC）

## 这是 H2 ← TOC 才认这个

### 这是 H3
```

---

### 问题 2：代码块样式不生效

**原因：** HTML 结构不符合预期（缺少 `<pre><code>` 嵌套）

**排查步骤：**
1. 打开浏览器开发工具 (F12)
2. 查看代码块 HTML 结构
3. 确保 `<pre><code class="language-xxx">` 格式

**临时修复：** 在 enhance.css 的代码块选择器后加 `!important`

---

### 问题 3：搜索功能找不到文章

**原因：** postList.json 未正确生成或文章未发布

**检查清单：**
- [ ] Issue 已发布（不是草稿）
- [ ] Actions 已运行完成（检查 Repository → Actions）
- [ ] 打开 https://yourblog.github.io/postList.json 查看是否包含新文章

---

### 问题 4：音效不播放

**原因：** 浏览器禁用 Web Audio / 用户关闭音效开关

**排查方法：**
1. 检查浏览器控制台是否有错误：`console.log(getAudioCtx())`
2. 查看 localStorage：`localStorage.getItem('luliy-sfx')`
3. 若为 '0'，点击音效图标长按 3 秒重新开启

---

### 问题 5：粒子背景卡顿

**原因：** Canvas 渲染性能不足（老电脑、移动设备）

**解决方案：**
```javascript
// 在 localStorage 中禁用
localStorage.setItem('luliy-particles', '0')
```

然后刷新页面，粒子背景会被禁用，性能显著改善

---

### 问题 6：暗色模式样式错乱

**原因：** 浏览器缓存或 CSS 加载顺序

**快速修复：**
```javascript
// 浏览器控制台运行
location.reload(true)  // 强制刷新（忽略缓存）
```

若仍未解决，检查 enhance.css 是否正确上传（网络标签查看）

---

### 问题 7：头像时钟不显示

**原因：** 头像元素不符合选择器，或 JS 未找到

**排查方法：**
1. 浏览器开发工具搜索 `.avatar` 或 `.blogTitle img`
2. 检查是否存在这些 class/结构
3. 若不存在，修改 initAvatarClock 中的选择器
